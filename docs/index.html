// Meet your buddy

Three poses, one job
→ remind you to take care.

Each timer has its own pose so the reminder feels personal — never just a beep.

Break

Stand up & stretch. Buddy is ready to move.

Water

Stay hydrated. Buddy carries the bottle for you.

Screen Break

Rest your eyes. Buddy hugs the cup, eyes closed.

// What's inside

Built to be ignored
only when you want to.

01 // Reminders

Three independent timers

Break, water, and screen-break — each with its own on/off switch and interval. Defaults are 45 / 90 / 60 minutes; tune anything.

02 // Heads-up

Layered warnings

A polite "1 min" popup with the buddy, a live 15-second countdown, then a full-screen reminder. You will not miss it.

03 // Considerate

No autopilot

Timers stop after each reminder so you decide when to restart. The full-screen modal locks for 5 seconds so it isn't dismissed by reflex.

04 // Concurrent

Reminders queue, not collide

If multiple timers fire at once, they queue and play sequentially. Other timers pause while a reminder is on screen.

05 // Tray

Lives in your system tray

Hide the window any time — Pausely keeps running. Open from the tray or reset any timer from the tray menu.

06 // Private

Fully offline

No accounts. No telemetry. No network calls. Settings stay on your device, period.

// How a reminder unfolds

Quiet, then clear,
then unmissable.

T-60 SEC

The buddy says hi

A small popup appears in the corner: "Get ready — your break buddy is heading over." A blinking orange dot, no fuss.

T-15 SEC

Live countdown

The popup switches to a live 15-second countdown so you can wrap up the sentence you're typing.

T-0 SEC

Full-screen reminder

A soft chime plays. The buddy takes over the whole screen. Restart the timer or close — the close button unlocks after five seconds.
